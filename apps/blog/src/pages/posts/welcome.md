# Welcome to Minions Bundle: Content

We're excited to introduce **Minions Bundle: Content** — Content pipeline bundle — drafts, publications, channels, campaigns.

## What's Included

- **TypeScript SDK**: `npm install @minions-bundles-content/sdk`
- **Python SDK**: `pip install minions-bundles-content`
- **CLI**: `npm install -g @minions-bundles-content/cli`

Check out the [documentation](https://docs.minions.wtf) for a full guide.
