---
title: Installation
description: "Install Minions Bundle: Content packages"
---

## TypeScript / Node.js

```bash
npm install @mxn2020/minions-bundles-content-core minions-sdk
```

## Python

```bash
pip install minions-bundles-content
```

## CLI

```bash
npm install -g @mxn2020/minions-bundles-content-cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
